FCFS

First Come First Served (FCFS) is the simplest non-preemptive scheduling algorithm. It allocates the CPU in the order of arrival.

02

SJF

Shortest Job First (SJF) selects the process with the smallest execution time. It may preempt current tasks if a shorter job arrives.

Read More Demo

03

SRTF

Shortest Remaining Time First (SRTF) is a preemptive version of SJF that executes the process with the least remaining time.

Read More Demo

04

Priority

Priority scheduling selects tasks based on their priority level. Higher-priority processes are executed before lower ones.
